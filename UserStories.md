1. As a collector, I want an easy way to add the units I've purchased to easily manage my inventory.
2. As a seller, I want to be able to remove miniatures from my account that I don't own anymore.
3. As a collector, I want to keep track of the myriad of multiple units I collect.
4. As a buyer, I want to be able to wishlist certain products so I can buy them in the future.
5. As a wargamer, I want to view what upgrade cards come in a unit box to build an effective tabletop list.
6. As a wargamer, I want to be able to search for specific cards in unit boxes I purchased to see if I own them already.
7. As a miniature painter, I want to be able to see example color schemes of units to get inspired.
8. As a wargamer, I want to be able to view the most popular competitive lists to build a powerful army.
9. As a collector, I want to be able to filter my invetory view to see different unit types and factions.
10. As a buyer, I want to see prices for units so I can create an accurate budget.
11. As a player, I want to be able to simulate format specific battles between armies to improve my listbuilding abilities.
12. As a player, I want to be able to simulate combat between individual units to inform my in-game decision making.
13. As a statistician, I want to be able to view unit and list winrates to determine the best units to use.
14. As a competitive player, I would like to know a unit's best and worst matchups by rank to avoid or pursue certain matchups while playing.
15. As a competitive player, I would like to know the best upgrades to take on units to maximize my winrate.
16. As a defensive player, I want to be able to view the ‘staying power’ of a unit, determining which units can survive the longest. 
17. As a tabletop strategist, I want to be able to have units spend tokens and perform actions like they would be able to in the physical wargame, making my practice easily transferable to a real game.
18. As a collector, I want to be able to build lists out of units that I own, so I can recreate winning lists without spending an exorbitant amount of money.
